# Claude Plugins

Organization plugin marketplace for Claude Code.

## Setup

```bash
/plugin marketplaces add https://github.com/RyotaSugawara/claude-plugins
```

## Plugins

| Plugin | Commands | Description |
|--------|----------|-------------|
| gh-dev | `/gh-dev:resume`, `/gh-dev:status`, `/gh-dev:complete` | GitHub Project & Milestone based development workflow |

## Install a plugin

```bash
/plugin install gh-dev@claude-plugins
```
